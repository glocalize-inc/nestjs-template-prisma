## Description

Backend template for NestJS with prisma

## Installation

```bash
$ npm i
$ prisma generate
$ cp .env.example .env
$ vi .env  # Sync your development(local) database
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
